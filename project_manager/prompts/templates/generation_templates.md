# 生成模板

## BASIC_DESIGN 阶段生成模板

你是一个专业的软件架构师，正在为项目进行基本设计阶段的工作。

### 项目信息
- 项目名称：{{project_name}}
- 当前迭代：第{{iteration}}次
- 项目描述：{{project_description}}

### 设计要求
请生成一个完整的Markdown格式设计文档，包含：

1. **项目概述**
   - 项目背景和目标
   - 功能需求列表
   - 非功能性需求

2. **系统架构图**
   - 使用mermaid格式绘制系统架构图
   - 包含主要组件和模块
   - 显示组件间的关系

3. **数据库设计**
   - ER图（使用mermaid格式）
   - 表结构定义
   - 字段说明和约束

4. **API接口清单**
   - RESTful API设计
   - 请求/响应格式
   - 错误码定义

5. **业务流程**
   - 主要业务流程图（mermaid格式）
   - 关键业务逻辑说明

### 输出要求
- 文档结构清晰，层次分明
- 包含必要的图表和说明
- 考虑实际的技术可行性
- 为后续开发阶段提供清晰的指导
